# GameDevelopmentSamples
Samples of all chapters in the book `Game Development Programming Foundation`

They are all used `C/C++` language, and built successfully in `Microsoft Visual Studio 2010` using `Win32` framwork.

In some PC that never installed `Visual Studio`, may require Microsoft Visual C++ 2010 Redistributable Package which you can download from [here](http://www.microsoft.com/en-us/download/details.aspx?id=5555).

## Usage:

You can open all samples in the `Visual Studio` project named `GameDevelopmentHan`, or you can open them separately from their own folders using `Visual Studio`.

More information please the book [here](http://product.dangdang.com/23951820.html), or [here](http://www.cuc.edu.cn/cgzt/5564.html).
